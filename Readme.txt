git commands
~~~~~~~~~~~~
git branch: 
    to list local repositories
git status:
    checks for stages files
    checks for committed files
git add .:
    add your changes to staging area
git commit -m 'message':
    add your changes to commit area
    in your local repository you are commiting the changes
git push origin branch_name:
    pushes your branch to remote repository 
git pull origin branch_name:
    pull the remote repository changes to local branch_name
